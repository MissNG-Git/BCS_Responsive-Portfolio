MissNG WeatherDash

GITHUB REPO

If you're looking to find an informative weather dashboard that displays not only the current weather but a 5-day forecast as well, then look no further! The MissNG WeatherDash is a simple, eye-catching weather dashboard I created for the explicit purpose of displaying up-to-date weather data to any user of this application.

It's a project I am proud of accomplishing considering it was my first try at utilising server-side APIs and incorporating relevant icons and Fahrenheit temperature conversion.


** TECHNOLOGIES USED: **
HTML, CSS & Javascript w/jQuery | Bootstrap | Open Weather Map API
